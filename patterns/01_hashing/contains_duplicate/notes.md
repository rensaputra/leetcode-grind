# Contains Duplicate

## Problem Statement

Given an integer array `nums`, return `true` if any value appears **at least twice** in the array, and return `false` if every element is distinct.

## Example

```Input: nums = [1,2,3,1]
Output: true
Explanation: The value 1 appears twice in the array, so we return true.
```

```Input: nums = [1,2,3,4]
Output: false
Explanation: Every element is distinct in the array, so we return false.
```
